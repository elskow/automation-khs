# Kartu Hasil Studi bot (KHS bot)

## How to run
1. Clone this repo
2. Make sure you have [PDM](https://pdm-project.org/) installed and [Chrome](https://www.google.com/chrome/) installed
3. Install dependencies
    ```bash
    pdm install
    ```
4. Run the bot
    ```bash
    pdm run dev
    ```
    > Yes it's still in development.
